# Excel-End-To-End-project
Interactive Superstore Sales Dashboard built in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, KPI Cards, and Data Visualization techniques to analyze sales performance, transactions, customer segments, regions, and monthly trends.

I'm happy to share my Superstore Sales Dashboard created entirely in Microsoft Excel.

This dashboard provides insights into:
✅ Total Sales
✅ Sales Average
✅ Total Transactions
✅ Total Quantity Sold
✅ Ship Mode Analysis
✅ Region-wise Performance
✅ Monthly Quantity Trend
✅ Top 5 Cities by Sales
✅ Category-wise Transaction Count
✅ Customer Segment Analysis

Through this project, I strengthened my skills in:
🔹 Pivot Tables & Pivot Charts
🔹 Slicers for Interactive Filtering
🔹 Dashboard Design & Layout
🔹 Data Visualization
🔹 KPI Cards
🔹 Excel Formatting & Data Analysis
